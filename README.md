# Bellabeat-Fitness-Analysis
Bellabeat Data Analysis: Transforming Fitness Tracker Insights into Growth Strategies. > A comprehensive SQL and Power BI project analyzing 12 months of smart device data to optimize wellness marketing. Features data cleaning, user segmentation, and trend visualization.


## 📌 Project Overview
This project analyzes smart device fitness data to identify growth opportunities for **Bellabeat**, a high-tech manufacturer of health-focused products for women. By examining user activity, sleep patterns, and engagement levels, this study provides data-driven recommendations to enhance marketing strategies and product features.

## 🛠️ Tools & Technologies
* **SQL (MySQL):** Data cleaning, transformation, and complex joins.
* **Power BI:** Interactive dashboarding and trend visualization.
* **Excel:** Initial data exploration and validation.

## 📊 Dataset
The analysis utilizes the **FitBit Fitness Tracker Data** (Kaggle). This dataset contains personal fitness logs from 30 users, including minute-level output for physical activity, heart rate, and sleep monitoring.

## ⚙️ Data Workflow
1. **Cleaning:** Standardized date formats and handled null values using MySQL.
2. **Transformation:** Aggregated minute-level sleep data into daily totals to match activity logs.
3. **Integration:** Created a unified dataset joining activity metrics (steps, calories) with sleep duration.
4. **Analysis:** Segmented users based on activity intensity (e.g., Sedentary vs. Highly Active).

## 💡 Key Insights
* **User Segmentation:** 46% of users are classified as 'Inactive' (under 5k steps/day), representing a significant target market for engagement features.
* **Temporal Trends:** Activity and calorie burn consistently peak mid-week (Tuesday–Friday), with a noticeable decline on weekends.
* **Sleep Correlation:** Inactive users recorded a higher average sleep duration (721 minutes) compared to more active groups.
* **Data Quality:** Only 18.75% of records included sleep data, highlighting a need for improved user retention in sleep tracking.

## 🚀 Strategic Recommendations
* **"Weekend Warrior" Campaigns:** Implement push notifications and challenges specifically for Saturdays and Sundays to combat the weekend activity dip.
* **Product Positioning:** Market the **Bellabeat Leaf** as a primary tool for "small daily wins," targeting sedentary users to bridge the gap to light activity.
* **Incentivized Logging:** Offer rewards or health "streaks" to encourage 24/7 device wear, specifically to improve sleep data coverage.
